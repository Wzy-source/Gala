# AURA
## Address
https://etherscan.io/address/0x967d176328948e4db4446b8caf623ff9b47221fb#code

## Vulnerability Description
Anyone can call `Owned.setOwner` function due to lack of `onlyOwner` modifierï¼Œcausing ownership of AURA to be transferred.


## Vulnerability Code
```solidity
contract Owned {
  address public owner;
  function Owned() public {
    owner = msg.sender;
  }
  function setOwner(address _owner) public returns (bool success) {
    owner = _owner;
    return true;
  }
  modifier onlyOwner {
    require(msg.sender == owner);
    _;
  }
}
```
